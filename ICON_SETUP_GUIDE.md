# Android Icon Setup Guide

## Icon Types and Where They Appear

### 1. Launcher Icon (Home Screen)
- **Location**: User's home screen and app drawer
- **Generated by**: flutter_launcher_icons
- **Config file**: flutter_launcher_icons.yaml
- **Best practices**:
  - Simple, clean design
  - Works well at small sizes (48x48 dp)
  - No text or minimal text
  - High contrast

### 2. Play Store Icon
- **Location**: Google Play Store listing
- **Uploaded via**: Google Play Console
- **Size**: 512x512 PNG
- **Best practices**:
  - Can be more detailed
  - Can include app name/tagline
  - Marketing-focused design
  - Eye-catching at larger sizes

### 3. Adaptive Icon Components (Android 8.0+)
- **Foreground**: Main icon element (108x108 dp, safe zone 66x66 dp)
- **Background**: Color or image behind foreground
- **System masks**: Circle, squircle, rounded square, etc.

## Current Configuration

```yaml
# flutter_launcher_icons.yaml
flutter_launcher_icons:
  android: "ic_launcher"
  ios: false

  # Home screen icon
  image_path: "assets/images/logo_transparent.png"

  # Adaptive icon for Android 8.0+
  adaptive_icon_background: "#1A1A2E"
  adaptive_icon_foreground: "assets/images/logo_transparent.png"
```

## How to Update Icons

### Step 1: Prepare Your Icons
1. **Launcher icon** (home screen):
   - Size: 1024x1024 PNG minimum
   - Transparent background recommended
   - Simple design that scales well

2. **Play Store icon**:
   - Size: 512x512 PNG exactly
   - Can have background
   - More detailed/promotional design

### Step 2: Update Launcher Icon
```bash
# 1. Place your new icon in assets/images/
# 2. Update flutter_launcher_icons.yaml
# 3. Run:
flutter pub run flutter_launcher_icons
```

### Step 3: Update Play Store Icon
1. Save your 512x512 icon as `app_store_assets/icons/playstore_icon_512.png`
2. Upload to Google Play Console:
   - Go to Play Console > Your App > Store presence > Store listing
   - Under "Graphics" section
   - Upload as "App icon"

## Design Tips

### For Launcher Icon:
- ✅ Bold, simple shapes
- ✅ High contrast
- ✅ Recognizable silhouette
- ❌ Avoid small text
- ❌ Avoid thin lines
- ❌ Avoid complex gradients

### For Play Store Icon:
- ✅ App name/tagline
- ✅ Detailed artwork
- ✅ Marketing appeal
- ✅ Brand colors
- ✅ Shadows/effects
- ✅ Background scenes

## Example Setup for Different Designs

```yaml
# flutter_launcher_icons.yaml for simple home screen icon
flutter_launcher_icons:
  android: "ic_launcher"

  # Simple icon for home screen
  image_path: "assets/images/logo_simple.png"

  # Adaptive icon with solid color background
  adaptive_icon_background: "#1A1A2E"
  adaptive_icon_foreground: "assets/images/logo_simple_foreground.png"
```

Then upload `logo_detailed_playstore.png` (512x512) to Play Console.

## Testing Your Icons

1. **Test launcher icon**:
   ```bash
   flutter run
   # Check home screen and app drawer
   ```

2. **Preview adaptive icon**:
   - Use Android Studio's Image Asset Studio
   - Or online tool: https://adapticon.tooo.io/

3. **Test Play Store appearance**:
   - Upload to Play Console as draft
   - Preview before publishing
