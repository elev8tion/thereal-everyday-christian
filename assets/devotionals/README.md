## File Format

Each batch file contains an array of devotional objects following the 8-section format:

```json
[
  {
    "id": "dev_001",
    "date": "2025-01-01",
    "title": "Trading Worry for Peace",
    "openingScripture": {
      "reference": "Philippians 4:6-7",
      "text": "Full WEB verse text..."
    },
    "keyVerseSpotlight": {
      "reference": "Philippians 4:6",
      "text": "Full WEB verse text..."
    },
    "reflection": "250-400 word reflection in 3 paragraphs...",
    "lifeApplication": "Because [truth], I will [action]...",
    "prayer": "Conversational, Scripture-rooted prayer...",
    "actionStep": "Specific measurable action for today...",
    "goingDeeper": [
      "Reference 1 - Description",
      "Reference 2 - Description"
    ],
    "readingTime": "3 min"
  }
]
```

   - `key_verse_text` (TEXT)
   - `reflection` (TEXT)

4. **Run Migration**: Execute migration to populate database with all 365 devotionals
